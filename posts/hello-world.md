I spend most of my time reading other people's code — database internals,
big-data platforms, desktop clients — looking for the places where it breaks.
Most of what I learn evaporates unless I write it down, so this is where it goes:
notes on crashes I've chased, systems I've taken apart, and the occasional
kernel benchmark.

Posts here will be short and technical. If you want the shorter version of what
I'm up to, the [home page](/) has the current list, and the
[findings](/proofs) page has proof of the fun ones.

More soon.
